# Text-Data-Representation-and-Vectorization
If you work with text data, you will always need this code. Here you will find One Hot Encodings, Bag of Words (BOW), N-Grams, Term frequency- Inverse document frequency(TF-IDF), word2vec.

You can access data: https://www.kaggle.com/datasets/khulasasndh/game-of-thrones-books
